# 🐍 Rope — Инструмент «Верёвка»

## Что мы делаем?

Создаём файл `Rope.cs` — инструмент, который **соединяет два объекта верёвкой**. Верёвка может быть гибкой (провисающей) или жёсткой, с настраиваемой длиной и толщиной.

## Зачем это нужно?

Rope — один из самых используемых инструментов в песочнице:
- Привязать объект к стене (качели, маятник)
- Связать два объекта между собой
- Создать подвеску для машины
- Декоративные верёвки и канаты

## Как это работает внутри движка?

### Атрибуты класса

```csharp
[Icon( "🐍" )]          — иконка в меню инструментов
[ClassName( "rope" )]    — имя для сети и TypeLibrary
[Group( "Constraints" )] — группа в меню
```

### Настройки (Properties)

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `Slack` | `float` | `0` | Провисание: положительное = длиннее, отрицательное = короче |
| `Rigid` | `bool` | `false` | Жёсткая верёвка (не провисает) |
| `Radius` | `float` | `1.0` | Толщина верёвки |

### Что создаётся?

При соединении двух точек создаются:

```
point1.GameObject
  └── go1 (child "rope")
        ├── ConstraintCleanup  — для системы очистки (undo, удаление)
        ├── SpringJoint        — физическое соединение (если объекты разные)
        ├── VerletRope         — физическая симуляция провисания (если не Rigid)
        └── LineRenderer       — визуальное отображение верёвки

point2.GameObject
  └── go2 (child "rope")  — вторая точка крепления
```

### SpringJoint (пружинное соединение)

```csharp
joint.MinLength = Rigid ? len : 0;   // жёсткая = не сжимается
joint.MaxLength = len;                 // максимальная длина = расстояние + slack
joint.RestLength = len;                // длина покоя
joint.Frequency = 0;                   // без пружинности
joint.Damping = 0;                     // без затухания
joint.EnableCollision = true;          // объекты сталкиваются друг с другом
```

### VerletRope (визуальная физика верёвки)

Если верёвка **не жёсткая**, добавляется `VerletRope` — компонент, который симулирует провисание. Количество сегментов зависит от длины (максимум 48).

### LineRenderer (отрисовка)

```csharp
lineRenderer.Points = [go1, go2];                    // от точки 1 к точке 2
lineRenderer.Width = Radius;                          // толщина
lineRenderer.Texturing = ... "rope01.vmat" ...       // текстура каната
lineRenderer.Face = SceneLineObject.FaceMode.Cylinder;// цилиндрическая форма
```

### Undo

```csharp
var undo = Player.Undo.Create();
undo.Name = "Rope";
undo.Add( go1 );
undo.Add( go2 );
```

Оба объекта (`go1` и `go2`) добавляются в систему отмены, чтобы игрок мог отменить действие через Z.

## Создай файл

📄 `Code/Weapons/ToolGun/Modes/Rope.cs`

```csharp
﻿﻿
[Icon( "🐍" )]
[ClassName( "rope" )]
[Group( "Constraints" )]
public class Rope : BaseConstraintToolMode
{
	[Range( -500, 500 )]
	[Property, Sync]
	public float Slack { get; set; } = 0.0f;

	[Property, Sync]
	public bool Rigid { get; set; } = false;

	[Range( 0.5f, 5f ), Step( 0.5f )]
	[Property]
	public float Radius { get; set; } = 1f;

	public override string Description => Stage == 1 ? "#tool.hint.rope.stage1" : "#tool.hint.rope.stage0";
	public override string PrimaryAction => Stage == 1 ? "#tool.hint.rope.finish" : "#tool.hint.rope.source";
	public override string ReloadAction => "#tool.hint.rope.remove";

	public override bool CanConstraintToSelf => true;

	protected override IEnumerable<GameObject> FindConstraints( GameObject linked, GameObject target )
	{
		foreach ( var cleanup in linked.GetComponentsInChildren<ConstraintCleanup>( true ) )
		{
			if ( linked != target && cleanup.Attachment?.Root != target ) continue;
			var go = cleanup.GameObject;
			if ( go.GetComponent<SpringJoint>() is not null || go.GetComponent<VerletRope>() is not null )
				yield return go;
		}
	}

	protected override void CreateConstraint( SelectionPoint point1, SelectionPoint point2 )
	{
		var go1 = new GameObject( false, "rope" );
		go1.Parent = point1.GameObject;
		go1.LocalTransform = point1.LocalTransform;
		go1.LocalRotation = Rotation.Identity;

		var go2 = new GameObject( false, "rope" );
		go2.Parent = point2.GameObject;
		go2.LocalTransform = point2.LocalTransform;
		go2.LocalRotation = Rotation.Identity;

		var len = point1.WorldPosition().Distance( point2.WorldPosition() );
		len = MathF.Max( 1.0f, len + Slack );

		var cleanup = go1.AddComponent<ConstraintCleanup>();
		cleanup.Attachment = go2;

		//
		// If it's ourself - we want to create the rope, but no joint between
		//
		if ( point1.GameObject != point2.GameObject )
		{
			var joint = go1.AddComponent<SpringJoint>();
			joint.Body = go2;
			joint.MinLength = Rigid ? len : 0;
			joint.MaxLength = len;
			joint.RestLength = len;
			joint.Frequency = 0;
			joint.Damping = 0;
			joint.EnableCollision = true;
		}

		var splineInterpolation = 0;
		if ( !Rigid )
		{
			var vertletRope = go1.AddComponent<VerletRope>();
			vertletRope.Attachment = go2;

			const int maxSegmentCount = 48;
			// Maximum segment count, so long ropes don't exceed computation limits
			int segmentCount = Math.Min( maxSegmentCount, MathX.CeilToInt( len / 16f ) );

			vertletRope.SegmentCount = segmentCount;
			vertletRope.Radius = Radius;
			splineInterpolation = segmentCount > maxSegmentCount ? 8 : 4;
		}

		var lineRenderer = go1.AddComponent<LineRenderer>();
		lineRenderer.Points = [go1, go2];
		lineRenderer.Width = Radius;
		lineRenderer.Color = Color.White;
		lineRenderer.Lighting = true;
		lineRenderer.CastShadows = true;
		lineRenderer.SplineInterpolation = splineInterpolation;
		lineRenderer.Texturing = lineRenderer.Texturing with { Material = Material.Load( "materials/default/rope01.vmat" ), WorldSpace = true, UnitsPerTexture = 32 };
		lineRenderer.Face = SceneLineObject.FaceMode.Cylinder;

		go2.NetworkSpawn( true, null );
		go1.NetworkSpawn( true, null );

		var undo = Player.Undo.Create();
		undo.Name = "Rope";
		undo.Add( go1 );
		undo.Add( go2 );
	}
}
```

## Особенности

### CanConstraintToSelf = true

Верёвку можно привязать к **одному и тому же** объекту (например, две точки на одном пропе). В этом случае `SpringJoint` **не создаётся** (нет смысла соединять объект с самим собой), но `VerletRope` и `LineRenderer` всё равно добавляются для визуала.

### Описание меняется по стадии

```csharp
public override string Description => Stage == 1 ? "#tool.hint.rope.stage1" : "#tool.hint.rope.stage0";
```

Строки с `#` — это ключи локализации. На стадии 0 — «Выберите первую точку», на стадии 1 — «Выберите вторую точку».

## Что проверить?

1. Экипируйте тулган → выберите Rope
2. Нажмите на первый объект → луч, стадия 1
3. Нажмите на второй объект → верёвка появляется
4. R на объекте → верёвка удаляется
5. Z (Undo) → верёвка удаляется
