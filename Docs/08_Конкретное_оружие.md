# Этап 8: Конкретное оружие

## Цель этого этапа

Создать конкретные виды оружия: пистолет, автомат, дробовик, монтировку (ближний бой) и гранату. Каждое оружие — маленький класс, наследующий BaseBulletWeapon.

## Как создаётся конкретное оружие

Благодаря иерархии наследования, создать новое оружие **очень просто**. Большая часть логики уже в базовых классах. Нужно только указать параметры.

## Шаг 8.1: Пистолет (Glock)

Создай файл `Code/Weapons/GlockWeapon.cs`:

```csharp
/// <summary>
/// Пистолет Glock. Полуавтоматический, средний урон.
/// Наследует BaseBulletWeapon — вся логика стрельбы уже есть!
/// </summary>
public class GlockWeapon : BaseBulletWeapon
{
    // Настройки по умолчанию можно переопределить в префабе,
    // но можно задать и здесь:

    protected override void OnEnabled()
    {
        // Параметры пистолета
        FireRate = 0.15f;        // 6-7 выстрелов/сек
        Automatic = false;       // Полуавтоматический (один клик = один выстрел)
        ClipSize = 17;           // 17 патронов в магазине
        BaseDamage = 8f;         // Урон за пулю
        Spread = 0.02f;          // Небольшой разброс
        ReloadTime = 1.5f;       // 1.5 сек перезарядка
        
        base.OnEnabled();
    }

    // Тип удержания — пистолет (одна рука)
    public override CitizenAnimationHelper.HoldTypes HoldType 
        => CitizenAnimationHelper.HoldTypes.Pistol;
}
```

### Вот и всё! 

Благодаря наследованию, этот пистолет уже умеет:
- ✅ Стрелять при нажатии ЛКМ
- ✅ Наносить урон рейкастом
- ✅ Перезаряжаться на R
- ✅ Отображать прицел
- ✅ Работать по сети

## Шаг 8.3: Дробовик (Shotgun)

Создай файл `Code/Weapons/Shotgun/ShotgunWeapon.cs`:

```csharp
/// <summary>
/// Дробовик. Стреляет несколькими пулями одновременно. 
/// Огромный урон вблизи, слабый вдали.
/// </summary>
public class ShotgunWeapon : BaseBulletWeapon
{
    protected override void OnEnabled()
    {
        FireRate = 0.8f;          // Медленная стрельба
        Automatic = false;        // Полуавтомат
        ClipSize = 8;             // 8 патронов
        BaseDamage = 7f;          // Урон ЗА ОДНУ ПУЛЮ
        BulletsPerShot = 8;       // 8 пуль за выстрел → 7×8 = 56 урона макс
        Spread = 0.15f;           // Большой разброс (конус)
        ReloadTime = 0.5f;        // Время зарядки ОДНОГО патрона
        
        base.OnEnabled();
    }

    public override CitizenAnimationHelper.HoldTypes HoldType 
        => CitizenAnimationHelper.HoldTypes.Shotgun;
}
```

### Как работает дробовик:

```
Один выстрел дробовика:

         Дуло
          │
    ╔═════╪═════╗
    ║  • •│• •  ║   ← 8 пуль разлетаются конусом
    ║ •   │   • ║      (Spread = 0.15 = широкий конус)
    ║  •  │  •  ║
    ╚═════╪═════╝

Если попали 3 пули: 7 × 3 = 21 урона
Если попали все 8: 7 × 8 = 56 урона (смертельно!)
```

## Шаг 8.4: Монтировка (ближний бой)

Ближний бой работает иначе — короткий рейкаст перед игроком.

Создай файл `Code/Game/Weapon/MeleeWeapon/MeleeWeapon.cs`:

```csharp
/// <summary>
/// Оружие ближнего боя. Короткий рейкаст перед игроком.
/// </summary>
public class MeleeWeapon : BaseWeapon
{
    [Property] public float MeleeDamage { get; set; } = 25f;
    [Property] public float MeleeRange { get; set; } = 80f;
    [Property] public float SecondaryDamage { get; set; } = 50f;
    [Property] public float SecondaryRange { get; set; } = 60f;

    public MeleeWeapon()
    {
        UsesAmmo = false; // Монтировке патроны не нужны!
    }

    public override void OnPlayerUpdate( Player player )
    {
        if ( Input.Pressed( "attack1" ) )
        {
            PrimaryAttack();
        }
        else if ( Input.Pressed( "attack2" ) )
        {
            SecondaryAttack();
        }
    }

    [Rpc.Host]
    private void PrimaryAttack()
    {
        DoMeleeAttack( MeleeDamage, MeleeRange );
    }

    [Rpc.Host]
    private void SecondaryAttack()
    {
        DoMeleeAttack( SecondaryDamage, SecondaryRange );
    }

    private void DoMeleeAttack( float damage, float range )
    {
        var player = Owner;
        if ( !player.IsValid() ) return;

        var eyePos = player.EyeTransform.Position;
        var eyeDir = player.EyeTransform.Forward;

        // Короткий рейкаст
        var tr = Scene.Trace
            .Ray( eyePos, eyePos + eyeDir * range )
            .IgnoreGameObjectHierarchy( player.GameObject )
            .Run();

        if ( tr.Hit )
        {
            var damageable = tr.GameObject?.GetComponent<Component.IDamageable>();
            if ( damageable != null )
            {
                var info = new DamageInfo( damage, player.GameObject, GameObject );
                damageable.OnDamage( in info );
            }
        }

        // Звук удара
        MeleeEffects( tr.Hit );
    }

    [Rpc.Broadcast]
    private void MeleeEffects( bool hit )
    {
        Sound.Play( hit ? "melee_hit" : "melee_miss", WorldPosition );
    }
}
```

## Шаг 8.5: Граната

Граната — уникальное оружие: её бросают, и через время она взрывается.

Создай файл `Code/Weapons/HandGrenade/TimedExplosive.cs`:

```csharp
/// <summary>
/// Компонент таймера взрыва. Прикрепляется к брошенной гранате.
/// </summary>
public class TimedExplosive : Component
{
    [Property] public float FuseTime { get; set; } = 3f;
    [Property] public float ExplosionDamage { get; set; } = 150f;
    [Property] public float ExplosionRadius { get; set; } = 300f;
    
    public GameObject Thrower { get; set; }

    private TimeSince _timeSinceCreated;

    protected override void OnEnabled()
    {
        _timeSinceCreated = 0;
    }

    protected override void OnUpdate()
    {
        if ( !Networking.IsHost ) return;

        if ( _timeSinceCreated >= FuseTime )
        {
            Explode();
        }
    }

    private void Explode()
    {
        // Находим всех в радиусе взрыва
        var objects = Scene.FindInPhysics( 
            new Sphere( WorldPosition, ExplosionRadius ) );

        foreach ( var obj in objects )
        {
            var damageable = obj.GetComponent<Component.IDamageable>();
            if ( damageable == null ) continue;

            // Урон уменьшается с расстоянием
            var distance = (obj.WorldPosition - WorldPosition).Length;
            var fraction = 1f - (distance / ExplosionRadius);
            var damage = ExplosionDamage * Math.Max( fraction, 0 );

            var dmg = new DamageInfo( damage, Thrower, GameObject );
            dmg.Tags.Add( DamageTags.Explosion );
            damageable.OnDamage( in dmg );
        }

        // Эффект взрыва и уничтожение
        ExplodeEffects();
        GameObject.Destroy();
    }

    [Rpc.Broadcast]
    private void ExplodeEffects()
    {
        Sound.Play( "explosion", WorldPosition );
        // Частицы взрыва
    }
}
```

### Как работает урон от взрыва:

```
Граната взрывается в точке X

    [100 урон]   [75 урон]    [50 урон]    [0 урон]
        ●──────────●──────────●──────────●
      Центр     1/4 радиуса  1/2 радиуса  За пределами

Формула: урон = базовый × (1 - расстояние/радиус)
```

## Шаг 8.6: Создание префабов оружия

Для каждого оружия нужен **префаб** — файл `.prefab`. Создаётся в редакторе:

1. **File → New Prefab**
2. Добавь компонент оружия (например `GlockWeapon`)
3. Добавь `DroppedWeapon` — чтобы можно было выбросить
4. Добавь `Rigidbody` — для физики выброшенного оружия
5. Добавь `ModelRenderer` — 3D модель
6. Настрой параметры в Property

### Структура файлов оружия:

```
Assets/weapons/
├── glock/
│   ├── glock.prefab        ← Префаб (перетаскивается в инвентарь)
│   ├── v_glock.prefab      ← ViewModel (от первого лица)
│   └── w_glock.prefab      ← WorldModel (видят все)
├── m4a1/

    └── toolgun.prefab
```

## ✅ Проверка

После создания хотя бы одного оружия и его префаба:

1. **Зайди в игру** — оружие автоматически выдаётся при спавне
2. **Стрельба** — ЛКМ стреляет, пули наносят урон
3. **Перезарядка** — R перезаряжает
4. **Переключение** — цифры 1-6 меняют оружие
