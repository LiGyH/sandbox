# 📚 Пошаговое руководство: Создание Sandbox с нуля

## Для кого это руководство?

Это руководство создано для тех, кто **никогда раньше не программировал** и хочет научиться создавать игры в движке **s&box**. Мы будем воссоздавать игру Sandbox (аналог Garry's Mod) **с нуля**, шаг за шагом.

## Как пользоваться этим руководством?

1. **Читай по порядку** — каждый этап строится на предыдущем
2. **Применяй изменения сразу** — после каждого шага ты можешь запустить проект и проверить результат
3. **Не торопись** — лучше понять каждый шаг, чем бежать вперёд

## Архитектура проекта (обзор)

```
sandbox/
├── sandbox.sbproj          ← Файл проекта s&box
├── ProjectSettings/        ← Настройки (ввод, физика, сеть, звук)
├── Assets/
│   ├── scenes/             ← Сцены (уровни)
│   ├── prefabs/            ← Готовые шаблоны объектов
│   ├── weapons/            ← Ресурсы оружия
│   └── ...                 ← Модели, звуки, материалы
├── Code/                   ← Весь код игры (C#)
│   ├── Player/             ← Система игрока
│   ├── GameLoop/           ← Менеджер игры
│   ├── Game/               ← Оружие, сущности, контроль
│   ├── Weapons/            ← Конкретное оружие
│   ├── Npcs/               ← NPC
│   ├── Spawner/            ← Система спавна
│   ├── UI/                 ← Интерфейс
│   ├── Map/                ← Объекты карты
│   ├── Save/               ← Сохранение
│   ├── Components/         ← Общие компоненты
│   ├── Utility/            ← Вспомогательные инструменты
│   ├── Cleanup/            ← Система очистки
│   └── FreeCam/            ← Свободная камера
└── Editor/                 ← Расширения редактора
```

> 💡 **Совет**: Перед тем как начать, прочитай **Этап 0** полностью. Там объясняются базовые понятия, без которых дальнейшие шаги будут непонятны.
