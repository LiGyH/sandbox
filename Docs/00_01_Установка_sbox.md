# 00.01 — Установка s&box

## Что мы делаем?

Устанавливаем движок **s&box** — это среда разработки игр от компании Facepunch (создатели Garry's Mod).  
s&box работает на движке **Source 2** от Valve (тот же движок, что у Dota 2 и Half-Life: Alyx).

## Что нужно?

1. **Компьютер с Windows 10/11** (64-bit)
2. **Steam** — установлен и запущен
3. **Интернет** — для скачивания

## Шаг 1: Установка Steam

Если у тебя ещё нет Steam:

1. Открой браузер, зайди на https://store.steampowered.com/
2. Нажми «Установить Steam»
3. Скачай и установи
4. Создай аккаунт (если нет)

## Шаг 2: Установка s&box

1. Открой **Steam**
2. В поиске набери **s&box**
3. Нажми **«Установить»** (игра бесплатная)
4. Дождись завершения загрузки (~2-5 ГБ)

> **Как это работает внутри движка:**  
> s&box при установке скачивает **бинарные файлы движка** (скомпилированный C++ код Source 2),  
> **редактор** (визуальный интерфейс для работы с проектами),  
> и **шаблоны проектов** (заготовки для новых игр).  
> Всё это можно увидеть в исходниках движка: https://github.com/LiGyH/sbox-public — папка `game/`.

## Шаг 3: Первый запуск

1. Запусти **s&box** через Steam
2. При первом запуске движок может скачать дополнительные данные
3. Ты увидишь **главное меню** с кнопками:
   - **Play** — играть в готовые игры
   - **Create** — создать свою игру
   - **Settings** — настройки

## Что такое s&box изнутри?

s&box — это **платформа для создания игр**. Она работает так:

```
┌─────────────────────────────────────────┐
│              s&box Editor               │  ← Визуальный редактор
├─────────────────────────────────────────┤
│         Твой код на C# (.cs)            │  ← Ты пишешь код здесь
├─────────────────────────────────────────┤
│     Sandbox.Engine (API движка)         │  ← Движок предоставляет классы
├─────────────────────────────────────────┤
│         Source 2 (Valve)                │  ← Графика, физика, звук
└─────────────────────────────────────────┘
```

Когда ты пишешь код на C#, движок **компилирует** его прямо в редакторе.  
Тебе не нужно устанавливать Visual Studio или .NET SDK — всё встроено.

> **Ссылка на документацию движка:**  
> https://sbox.game/dev/doc/about/getting-started/first-steps/

- ✅ Установлен s&box
- ✅ Ты можешь запустить редактор
