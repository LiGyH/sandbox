# Этап 05_03 — Chat (Чат)

## Что это такое?

Представьте групповой чат в мессенджере: вы видите сообщения других людей, можете написать своё, а старые сообщения постепенно исчезают. **Chat** — это встроенный чат в игре, где игроки общаются текстом прямо во время матча.

---

## Файл `Chat.razor`

### Полный исходный код

**Путь:** `Code/UI/Chat.razor`

```razor
@using Sandbox;
@using Sandbox.UI;
@using Sandbox.Utility
@inherits PanelComponent

<root>

    <div class="output">
        @foreach (var entry in Entries)
        {
            <div class="chat_entry">
                @if (entry.steamid > 0)
                {
                    <div class="avatar" style="background-image: url( avatar:@entry.steamid )"></div>
                }
                <div class="author">@entry.author</div>
                <div class="message">@entry.message</div>
            </div>
        }
    </div>

    <div class="input">
        <TextEntry @ref="InputBox" onsubmit="@ChatFinished"></TextEntry>
    </div>

</root>

@code
{
    TextEntry InputBox = default;

    public record Entry( ulong steamid, string author, string message, RealTimeSince timeSinceAdded );
    List<Entry> Entries = new();

    protected override void OnUpdate()
    {
        if (InputBox is null)
            return;

        Panel.AcceptsFocus = false;

        if ( Input.Pressed( "chat" ) )
        {
            InputBox.Focus();
        }

        if ( Entries.RemoveAll( x => x.timeSinceAdded > 20.0f ) > 0 )
        {
            StateHasChanged();
        }

        SetClass( "open", InputBox.HasFocus );
        SetClass( "hide", Player.FindLocalPlayer()?.WantsHideHud ?? false );
    }

    void ChatFinished()
    {
        var text = InputBox.Text;
        InputBox.Text = "";

        if (string.IsNullOrWhiteSpace(text))
            return;

        AddText( text );
    }

    [ConCmd( "say" )]
    private static void Say( string msg )
    {
       var chat = Game.ActiveScene.GetAll<Chat>()
                                .FirstOrDefault();

       if ( Application.IsDedicatedServer )
       {
           chat.AddSystemText( msg );
       }
       else
       {
           chat.AddText( msg );
       }
    }

	[Rpc.Broadcast]
    public void AddText( string message )
    {
        message = message.Truncate( 300 );

        if (string.IsNullOrWhiteSpace(message))
            return;

        var author = Rpc.Caller.DisplayName;
        var steamid = Rpc.Caller.SteamId;

		Log.Info($"{author}: {message}");

		if ( Rpc.Caller == Connection.Local && message.Contains( "colmore", StringComparison.OrdinalIgnoreCase ) )
		{
			Sandbox.Services.Stats.Increment( "secret_phrase", 1 );
        }

        var filteredName = Steam.FilterName(author, steamid);
        message = Steam.FilterChat(message, steamid);

        Entries.Add(new Entry(steamid, filteredName, message, 0.0f));
		StateHasChanged();
	}

	[Rpc.Broadcast( NetFlags.HostOnly )]
    public void AddSystemText( string message, string icon = "ℹ️" )
    {
        message = message.Truncate(300);

        if (string.IsNullOrWhiteSpace(message))
            return;

        Entries.Add(new Entry(0, icon, message, 0.0f));
        StateHasChanged();
    }

}
```

### Разбор кода

**Разметка:**

- `<div class="output">` — область вывода сообщений
- `@foreach (var entry in Entries)` — перебирает все сообщения и отображает каждое
- `avatar:@entry.steamid` — загружает аватар игрока из Steam по его ID
- `<TextEntry>` — поле ввода текста. `@ref="InputBox"` сохраняет ссылку на него, `onsubmit` — что делать при отправке

**Модель данных:**

```csharp
public record Entry( ulong steamid, string author, string message, RealTimeSince timeSinceAdded );
```
Каждое сообщение — это запись (record) с полями:
- `steamid` — числовой ID автора в Steam
- `author` — имя автора
- `message` — текст сообщения
- `timeSinceAdded` — сколько прошло с момента добавления

**Основные методы:**

| Метод | Что делает |
|-------|-----------|
| `OnUpdate()` | Каждый кадр: открывает чат по нажатию клавиши, удаляет сообщения старше 20 секунд |
| `ChatFinished()` | Вызывается при нажатии Enter: берёт текст, очищает поле ввода, отправляет сообщение |
| `Say()` | Консольная команда `say` — позволяет отправить сообщение из консоли |
| `AddText()` | Отправляет сообщение **всем игрокам** (атрибут `[Rpc.Broadcast]`). Фильтрует нецензурные слова через Steam |
| `AddSystemText()` | Системное сообщение (только от хоста сервера). Например, «Игрок был кикнут» |

**Важные детали:**

- `message.Truncate(300)` — обрезает сообщение до 300 символов (защита от спама)
- `Steam.FilterName()` и `Steam.FilterChat()` — фильтрация ненормативной лексики через Steam
- `Entries.RemoveAll( x => x.timeSinceAdded > 20.0f )` — сообщения автоматически исчезают через 20 секунд
- `[Rpc.Broadcast]` — метод вызывается на всех компьютерах в сети (мультиплеер)

## Как работает чат

```
1. Игрок нажимает клавишу "chat"
2. Поле ввода получает фокус (InputBox.Focus())
3. Игрок печатает текст и нажимает Enter
4. ChatFinished() берёт текст и вызывает AddText()
5. [Rpc.Broadcast] отправляет сообщение ВСЕМ игрокам
6. На каждом компьютере сообщение добавляется в список Entries
7. Через 20 секунд сообщение автоматически удаляется
```
