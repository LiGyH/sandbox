# 📘 Sandbox Game — Полная документация

## Как пользоваться этой документацией

Документы пронумерованы **ФФ_НН** (фаза_номер) и расположены в порядке, в котором **рекомендуется читать и реализовывать**. Каждый файл начинает с «Что мы делаем?» и объясняет код на русском языке с примерами.

> **Принцип**: Внедряй по очереди, проверяй после каждого шага. Каждый файл — самодостаточный этап.

2. **Фаза 2**: Интерфейсы — основа для всех систем
3. **Фаза 3**: Игрок — центральный элемент
4. **Фаза 4**: GameManager — цикл игры
5. **Фаза 5**: Компоненты — верёвки, стулья, колёса
6. **Фазы 6-7**: Оружие — от абстракции к конкретике
7. **Фаза 8**: Physgun/Toolgun — ключевые инструменты Sandbox
8. **Фаза 9**: Спавнеры — как появляются объекты
9. **Фаза 10**: UI — спавн-меню, инвентарь
10. **Фаза 11**: Игровые системы — баны, управление, пост-обработка
11. **Фаза 12**: Элементы карты — двери, кнопки, лифты, триггеры
12. **Фаза 13**: Предметы — пикапы, аптечки, патроны
13. **Фаза 14**: NPC — от ядра к конкретным типам
14. **Фаза 15**: Сохранение и очистка
15. **Фаза 16**: Свободная камера
