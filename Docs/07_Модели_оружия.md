# Этап 7: Модели оружия — ViewModel и WorldModel

## Цель этого этапа

Понять и создать систему отображения оружия:
- **ViewModel** — модель от первого лица (видишь только ты)
- **WorldModel** — модель в мире (видят все остальные)

## Две модели одного оружия

Когда ты держишь пистолет в игре, на самом деле существуют **две отдельные модели**:

```
Что видишь ТЫ:
┌─────────────────────────────────────┐
│                                     │
│         Мир, враги, карта           │
│                                     │
│                  +                  │  ← Прицел (HUD)
│                                     │
│                          ┌───┐      │
│                          │🔫 │      │  ← ViewModel (крупная, детальная)
│                          └───┘      │
└─────────────────────────────────────┘

Что видят ДРУГИЕ игроки:
┌─────────────────────────────────────┐
│                                     │
│              🧑🔫                    │  ← WorldModel (маленькая, в руке)
│                                     │
└─────────────────────────────────────┘
```

### Почему две модели?

1. **ViewModel** — детальная, анимированная, занимает много экрана. Рисуется с другим FOV (поле зрения), чтобы не искажаться по краям.
2. **WorldModel** — прикреплена к скелету персонажа, видна всем. Менее детальная.

## Шаг 7.1: WeaponModel — базовый класс

Создай файл `Code/Game/Weapon/WeaponModel/WeaponModel.cs`:

```csharp
/// <summary>
/// Базовый класс для моделей оружия (и ViewModel, и WorldModel).
/// Содержит общие данные: точка вылета пуль (muzzle), анимации.
/// </summary>
public class WeaponModel : Component
{
    /// <summary>
    /// Точка, из которой "вылетают" пули (дуло).
    /// Используется для трассеров и вспышек.
    /// </summary>
    [Property] public GameObject MuzzleTransform { get; set; }

    /// <summary>
    /// Точка выброса гильз.
    /// </summary>
    [Property] public GameObject EjectionTransform { get; set; }

    /// <summary>
    /// Рендерер модели.
    /// </summary>
    public SkinnedModelRenderer Renderer => GetComponent<SkinnedModelRenderer>();

    /// <summary>
    /// Установить параметр анимации.
    /// </summary>
    public void SetAnimParameter( string name, bool value )
    {
        Renderer?.Set( name, value );
    }

    public void SetAnimParameter( string name, int value )
    {
        Renderer?.Set( name, value );
    }

    public void SetAnimParameter( string name, float value )
    {
        Renderer?.Set( name, value );
    }
}
```

## Шаг 7.2: ViewModel — модель от первого лица

Создай файл `Code/Game/Weapon/WeaponModel/ViewModel.cs`:

```csharp
/// <summary>
/// Модель оружия от первого лица. Видна только локальному игроку.
/// Рисуется поверх всего с отдельным FOV.
/// </summary>
public partial class ViewModel : WeaponModel
{
    /// <summary>
    /// Поле зрения для оружия (обычно меньше, чем у камеры).
    /// Это нужно чтобы оружие не искажалось на широких мониторах.
    /// </summary>
    [Property] public float FieldOfView { get; set; } = 60f;

    /// <summary>
    /// Инерция — оружие слегка отстаёт от движения камеры.
    /// Создаёт ощущение веса.
    /// </summary>
    [Property] public float SwayAmount { get; set; } = 1.0f;

    /// <summary>
    /// Амплитуда покачивания при ходьбе.
    /// </summary>
    [Property] public float BobAmount { get; set; } = 1.0f;

    private Vector3 _swayOffset;
    private Angles _swayAngles;

    protected override void OnUpdate()
    {
        if ( !IsProxy ) return;

        var player = GetComponentInParent<Player>( true );
        if ( !player.IsValid() || !player.IsLocalPlayer ) return;

        // Инерция — оружие "запаздывает" за камерой
        var mouseDelta = Input.MouseDelta * Time.Delta;
        _swayAngles += new Angles( -mouseDelta.y, -mouseDelta.x, 0 ) * SwayAmount;
        _swayAngles = Angles.Lerp( _swayAngles, Angles.Zero, Time.Delta * 10f );

        // Покачивание при ходьбе
        var controller = player.Controller;
        if ( controller.IsValid() && controller.IsOnGround )
        {
            var speed = controller.Velocity.Length;
            var bob = MathF.Sin( Time.Now * 8f ) * speed * 0.0001f * BobAmount;
            _swayOffset = Vector3.Lerp( _swayOffset, new Vector3( 0, 0, bob ), Time.Delta * 10f );
        }

        // Применяем покачивание
        LocalPosition += _swayOffset;
        LocalRotation *= _swayAngles.ToRotation();
    }
}
```

### Как работает инерция оружия (Sway):

```
Камера поворачивается вправо:
Кадр 1: Камера ──→  Оружие ·     (оружие отстаёт)
Кадр 2: Камера ──→  Оружие ─→    (догоняет)
Кадр 3: Камера ──→  Оружие ──→   (почти догнало)
Кадр 4: Камера ──→  Оружие ──→   (совпало)

Это создаёт ощущение ВЕСА оружия.
```

## Шаг 7.3: WorldModel — модель в мире

Создай файл `Code/Game/Weapon/WeaponModel/WorldModel.cs`:

```csharp
/// <summary>
/// Модель оружия, видимая другим игрокам.
/// Прикреплена к скелету персонажа (кость руки).
/// </summary>
public class WorldModel : WeaponModel
{
    /// <summary>
    /// К какой кости прикрепить модель.
    /// "hold_r" — правая рука.
    /// </summary>
    [Property] public string AttachBone { get; set; } = "hold_r";

    protected override void OnEnabled()
    {
        // Устанавливаем тег, чтобы камера от первого лица не рисовала WorldModel
        Tags.Add( "world_model" );

        base.OnEnabled();
    }
}
```

## Шаг 7.4: Создание и уничтожение моделей

Добавь в `BaseCarryable` (как partial-файл или в основной файл) методы для управления моделями:

Создай файл `Code/Game/Weapon/BaseCarryable/BaseCarryable.ViewModel.cs`:

```csharp
public partial class BaseCarryable
{
    /// <summary>
    /// Префаб ViewModel (от первого лица).
    /// </summary>
    [Property, Feature( "Models" )] 
    public GameObject ViewModelPrefab { get; set; }

    /// <summary>
    /// Создаёт ViewModel. Только для локального игрока.
    /// </summary>
    protected void CreateViewModel()
    {
        if ( ViewModel.IsValid() ) return;
        if ( ViewModelPrefab is null ) return;

        var player = Owner;
        if ( !player.IsValid() || !player.IsLocalPlayer ) return;

        // Клонируем префаб
        ViewModel = ViewModelPrefab.Clone();
        ViewModel.Flags = GameObjectFlags.NotSaved;

        // ViewModel привязан к камере, а не к игроку
        var camera = Scene.Camera;
        if ( camera.IsValid() )
        {
            ViewModel.Parent = camera.GameObject;
        }
    }

    /// <summary>
    /// Уничтожает ViewModel.
    /// </summary>
    protected void DestroyViewModel()
    {
        if ( ViewModel.IsValid() )
        {
            ViewModel.Destroy();
            ViewModel = null;
        }
    }
}
```

Создай файл `Code/Game/Weapon/BaseCarryable/BaseCarryable.WorldModel.cs`:

```csharp
public partial class BaseCarryable
{
    /// <summary>
    /// Префаб WorldModel (видимый другим).
    /// </summary>
    [Property, Feature( "Models" )] 
    public GameObject WorldModelPrefab { get; set; }

    /// <summary>
    /// Создаёт WorldModel. Для всех игроков.
    /// </summary>
    protected void CreateWorldModel()
    {
        if ( WorldModel.IsValid() ) return;

        var player = Owner;
        if ( !player.IsValid() ) return;

        // Находим кость руки на модели персонажа
        var body = player.Body;
        if ( !body.IsValid() ) return;

        var renderer = body.GetComponent<SkinnedModelRenderer>();
        if ( !renderer.IsValid() ) return;

        if ( WorldModelPrefab is not null )
        {
            WorldModel = WorldModelPrefab.Clone();
            
            // Прикрепляем к кости руки
            var boneObj = renderer.GetBoneObject( "hold_r" );
            if ( boneObj.IsValid() )
            {
                WorldModel.Parent = boneObj;
                WorldModel.LocalPosition = Vector3.Zero;
                WorldModel.LocalRotation = Rotation.Identity;
            }
        }
    }

    /// <summary>
    /// Уничтожает WorldModel.
    /// </summary>
    protected void DestroyWorldModel()
    {
        if ( WorldModel.IsValid() )
        {
            WorldModel.Destroy();
            WorldModel = null;
        }
    }
}
```

### Как всё работает вместе:

```
Оружие ВКЛЮЧАЕТСЯ (OnEnabled):
    ↓
CreateWorldModel()  → клонирует модель → прикрепляет к руке
    ↓
Если это локальный игрок:
    CreateViewModel() → клонирует модель → прикрепляет к камере

Оружие ВЫКЛЮЧАЕТСЯ (OnDisabled):
    ↓
DestroyWorldModel() → удаляет модель из руки
DestroyViewModel()  → удаляет модель с камеры
```

### Почему ViewModel прикреплён к камере?

ViewModel рисуется в **экранном пространстве**, поверх всего мира. Он должен двигаться с камерой, а не с персонажем. Поэтому он привязан к `Scene.Camera.GameObject`.

```
Камера смотрит влево → ViewModel автоматически смотрит влево
Камера смотрит вверх → ViewModel автоматически смотрит вверх
```

## Шаг 7.5: HoldType — тип удержания

Каждое оружие указывает, как персонаж его держит. Это влияет на анимацию рук:

```csharp
// В BaseCarryable:
public virtual CitizenAnimationHelper.HoldTypes HoldType 
    => CitizenAnimationHelper.HoldTypes.None;
```

Типы удержания (из движка `Sandbox.Citizen`):
- `None` — руки свободны
- `Pistol` — пистолет (одна рука)
- `Rifle` — винтовка (две руки)
- `Shotgun` — дробовик
- `Item` — предмет
- `Fists` — кулаки
- `HoldItem` — держит что-то большое

Каждое конкретное оружие переопределяет `HoldType`:

```csharp
// В пистолете:
public override HoldTypes HoldType => HoldTypes.Pistol;

// В автомате:
public override HoldTypes HoldType => HoldTypes.Rifle;
```
