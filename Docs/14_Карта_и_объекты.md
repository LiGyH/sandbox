# Этап 14: Карта и объекты

## Цель этого этапа

Создать интерактивные объекты карты: двери, кнопки, платформы, телепорты, триггеры.

## Как карты работают в s&box

Карта загружается компонентом `MapInstance`. Она может содержать:
- Геометрию (стены, полы, потолки)
- Точки спавна (`SpawnPoint`)
- Интерактивные объекты (двери, кнопки)
- Триггеры (невидимые зоны)

Объекты карты — это **компоненты**, прикреплённые к GameObject'ам.

## Шаг 14.1: BaseToggle — базовый переключатель

Многие объекты карты имеют два состояния: включён/выключен, открыт/закрыт. Для них есть базовый класс.

Создай файл `Code/Map/BaseToggle.cs`:

```csharp
/// <summary>
/// Базовый переключаемый объект (дверь, кнопка, платформа).
/// Имеет два состояния: включён (On) и выключен (Off).
/// </summary>
public class BaseToggle : Component
{
    /// <summary>
    /// Текущее состояние. Синхронизируется по сети.
    /// </summary>
    [Sync] public bool IsToggled { get; set; } = false;

    /// <summary>
    /// Переключить состояние.
    /// </summary>
    public virtual void Toggle()
    {
        IsToggled = !IsToggled;
        OnToggled( IsToggled );
    }

    /// <summary>
    /// Включить.
    /// </summary>
    public virtual void TurnOn()
    {
        if ( IsToggled ) return;
        IsToggled = true;
        OnToggled( true );
    }

    /// <summary>
    /// Выключить.
    /// </summary>
    public virtual void TurnOff()
    {
        if ( !IsToggled ) return;
        IsToggled = false;
        OnToggled( false );
    }

    /// <summary>
    /// Вызывается при изменении состояния.
    /// </summary>
    protected virtual void OnToggled( bool state ) { }
}
```

## Шаг 14.2: Door — Дверь

Создай файл `Code/Map/Door.cs`:

```csharp
/// <summary>
/// Дверь. Открывается/закрывается при использовании (E).
/// Поворачивается или сдвигается в зависимости от настроек.
/// </summary>
public class Door : BaseToggle, Component.IInteractable
{
    /// <summary>
    /// Угол открытия двери (в градусах).
    /// </summary>
    [Property] public float OpenAngle { get; set; } = 90f;

    /// <summary>
    /// Скорость открытия.
    /// </summary>
    [Property] public float Speed { get; set; } = 2f;

    /// <summary>
    /// Звук открытия.
    /// </summary>
    [Property] public string OpenSound { get; set; } = "door_open";
    
    [Property] public string CloseSound { get; set; } = "door_close";

    private Rotation _closedRotation;
    private Rotation _openRotation;

    protected override void OnEnabled()
    {
        _closedRotation = LocalRotation;
        _openRotation = _closedRotation * Rotation.FromAxis( Vector3.Up, OpenAngle );
    }

    /// <summary>
    /// IInteractable — когда игрок нажимает E на двери.
    /// </summary>
    public void OnInteract( GameObject interactor )
    {
        Toggle();
    }

    protected override void OnToggled( bool state )
    {
        Sound.Play( state ? OpenSound : CloseSound, WorldPosition );
    }

    protected override void OnUpdate()
    {
        // Плавно поворачиваем дверь
        var targetRotation = IsToggled ? _openRotation : _closedRotation;
        LocalRotation = Rotation.Slerp( 
            LocalRotation, targetRotation, Time.Delta * Speed );
    }
}
```

### Как работает IInteractable:

`IInteractable` — интерфейс движка. Когда игрок смотрит на объект с этим интерфейсом и нажимает **E** (Use), движок вызывает `OnInteract()`.

## Шаг 14.3: Button — Кнопка

Создай файл `Code/Map/Button.cs`:

```csharp
/// <summary>
/// Кнопка. При нажатии отправляет событие связанным объектам.
/// </summary>
public class Button : BaseToggle, Component.IInteractable
{
    /// <summary>
    /// Время до автоматического выключения (0 = не выключается).
    /// </summary>
    [Property] public float ResetTime { get; set; } = 3f;

    /// <summary>
    /// Объекты, которые будут переключены при нажатии.
    /// </summary>
    [Property] public List<BaseToggle> Targets { get; set; } = new();

    private TimeSince _timeSincePressed;

    public void OnInteract( GameObject interactor )
    {
        if ( IsToggled ) return; // Уже нажата

        TurnOn();
        _timeSincePressed = 0;

        // Переключаем все связанные объекты
        foreach ( var target in Targets )
        {
            if ( target.IsValid() )
                target.Toggle();
        }
    }

    protected override void OnUpdate()
    {
        // Автоматический сброс
        if ( IsToggled && ResetTime > 0 && _timeSincePressed >= ResetTime )
        {
            TurnOff();
        }
    }
}
```

### Связи между объектами:

```
Кнопка ──(Target)──→ Дверь
                      ↓
              Дверь.Toggle() → Открывается!

Кнопка ──(Target)──→ Свет
                      ↓
              Свет.Toggle() → Включается!
```

## Шаг 14.4: FuncMover — Движущаяся платформа

Создай файл `Code/Map/FuncMover.cs`:

```csharp
/// <summary>
/// Движущаяся платформа. Перемещается между двумя точками.
/// </summary>
public class FuncMover : BaseToggle
{
    /// <summary>
    /// Смещение при активации (куда двигается).
    /// </summary>
    [Property] public Vector3 MoveOffset { get; set; } = new Vector3( 0, 0, 128 );

    /// <summary>
    /// Скорость перемещения.
    /// </summary>
    [Property] public float MoveSpeed { get; set; } = 100f;

    private Vector3 _startPosition;
    private Vector3 _endPosition;

    protected override void OnEnabled()
    {
        _startPosition = LocalPosition;
        _endPosition = _startPosition + MoveOffset;
    }

    protected override void OnUpdate()
    {
        var targetPos = IsToggled ? _endPosition : _startPosition;
        LocalPosition = Vector3.MoveTowards( 
            LocalPosition, targetPos, MoveSpeed * Time.Delta );
    }
}
```

## Шаг 14.5: TriggerTeleport — Телепорт

Создай файл `Code/Map/TriggerTeleport.cs`:

```csharp
/// <summary>
/// Триггер-телепорт. Когда объект входит в зону — 
/// телепортирует его в другую точку.
/// </summary>
public class TriggerTeleport : Component, Component.ITriggerListener
{
    /// <summary>
    /// Куда телепортировать (объект-цель).
    /// </summary>
    [Property] public GameObject Target { get; set; }

    /// <summary>
    /// ITriggerListener — вызывается когда что-то входит в триггер.
    /// </summary>
    void ITriggerListener.OnTriggerEnter( Collider other )
    {
        if ( !Target.IsValid() ) return;
        if ( !Networking.IsHost ) return;

        // Телепортируем
        var go = other.GameObject;
        go.WorldPosition = Target.WorldPosition;
        go.WorldRotation = Target.WorldRotation;
    }
}
```

### Что такое триггер?

**Триггер** — это невидимый коллайдер, через который можно **пройти**, но который **уведомляет** о входе/выходе.

```
Обычный коллайдер:     Триггер:
┌────────┐             ┌ ─ ─ ─ ─┐
│СТЕНА   │ ← не пройти   ЗОНА    ← можно пройти
│████████│             └ ─ ─ ─ ─┘
└────────┘             OnTriggerEnter() ← но получишь уведомление!
```

## Шаг 14.6: TriggerPush — Толкающий триггер

Создай файл `Code/Map/TriggerPush.cs`:

```csharp
/// <summary>
/// Триггер, который толкает объекты в указанном направлении.
/// Используется для: вентиляторы, потоки воды, трамплины.
/// </summary>
public class TriggerPush : Component, Component.ITriggerListener
{
    /// <summary>
    /// Направление и сила толчка.
    /// </summary>
    [Property] public Vector3 PushForce { get; set; } = new Vector3( 0, 0, 500 );

    void ITriggerListener.OnTriggerEnter( Collider other )
    {
        var body = other.GameObject?.GetComponent<Rigidbody>();
        if ( body.IsValid() )
        {
            body.Velocity += PushForce;
        }

        // Также толкаем игроков
        var controller = other.GameObject?.GetComponent<PlayerController>();
        if ( controller.IsValid() )
        {
            controller.Velocity += PushForce;
        }
    }
}
```
