# Этап 6: Базовое оружие

## Цель этого этапа

Создать иерархию классов оружия — от абстрактного «предмета в руках» до конкретного «пистолета, который стреляет». 

После этого этапа ты поймёшь, как работает система оружия, и сможешь создавать своё.

## Архитектура оружия — иерархия наследования

В Sandbox оружие строится слоями, каждый добавляет функциональность:

```
BaseCarryable                ← Базовый предмет (в руках, в инвентаре)
    │
    ├── BaseWeapon            ← Оружие (патроны, перезарядка, стрельба)
    │   │
    │   ├── BaseBulletWeapon  ← Стрелковое оружие (hitscan, трейсы)
    │   │   │
    │   │   └── Glock, Colt, MP5, M4A1, Shotgun
    │   │
    │   └── MeleeWeapon       ← Ближний бой (монтировка)
    │
    ├── Physgun              ← Физическая пушка (не наследует BaseWeapon)
    ├── Toolgun              ← Инструментальная пушка
    └── SpawnerWeapon        ← Оружие для спавна пропов
```

### Почему такая иерархия?

- **BaseCarryable** знает как быть в инвентаре, создавать модели, рисовать HUD
- **BaseWeapon** добавляет патроны, перезарядку, автоматическую стрельбу
- **BaseBulletWeapon** добавляет рейкаст (луч), попадания, урон

Physgun и Toolgun **не** наследуют BaseWeapon, потому что они не стреляют пулями.

## Шаг 6.1: BaseCarryable — базовый предмет

Это самый базовый класс. Любой предмет, который можно держать в руках, наследуется от него.

Создай файл `Code/Game/Weapon/BaseCarryable/BaseCarryable.cs`:

```csharp
using Sandbox.Rendering;

/// <summary>
/// Информация о трассировочной атаке (попадание пули/удар).
/// </summary>
public record struct TraceAttackInfo( 
    GameObject Target,     // Во что попали
    float Damage,          // Урон
    TagSet Tags = null,    // Теги урона
    Vector3 Position = default,  // Где попали
    Vector3 Origin = default     // Откуда стреляли
)
{
    public static TraceAttackInfo From( SceneTraceResult tr, float damage, 
        TagSet tags = default )
    {
        tags ??= new();
        return new TraceAttackInfo( tr.GameObject, damage, tags, 
            tr.HitPosition, tr.StartPosition );
    }
}

/// <summary>
/// Базовый класс для всех предметов в руках.
/// Управляет: инвентарным слотом, моделями (view/world), HUD.
/// </summary>
public partial class BaseCarryable : Component, IKillIcon
{
    // --- Настройки инвентаря ---
    
    [Property, Feature( "Inventory" )] 
    public string DisplayName { get; set; } = "My Weapon";
    
    [Property, Feature( "Inventory" ), TextArea] 
    public Texture DisplayIcon { get; set; }

    /// <summary>
    /// Значение оружия — для автопереключения.
    /// Оружие с большим Value будет выбрано автоматически.
    /// </summary>
    [Property, Feature( "Inventory" )] 
    public int Value { get; set; } = 0;

    // --- Модели ---
    
    /// <summary>
    /// ViewModel — модель от первого лица (видит только владелец).
    /// </summary>
    public GameObject ViewModel { get; protected set; }
    
    /// <summary>
    /// WorldModel — модель в мире (видят все остальные).
    /// </summary>
    public GameObject WorldModel { get; protected set; }

    /// <summary>
    /// Слот в инвентаре (0-5). -1 = не назначен.
    /// </summary>
    [Sync( SyncFlags.FromHost )] 
    public int InventorySlot { get; set; } = -1;

    /// <summary>
    /// Владелец этого предмета (игрок, у которого он в инвентаре).
    /// Ищет Player в родительских объектах.
    /// </summary>
    public Player Owner
    {
        get { return GetComponentInParent<Player>( true ); }
    }

    public bool HasOwner => Owner.IsValid();

    // --- Жизненный цикл ---

    /// <summary>
    /// Когда компонент включается — создаём мировую модель.
    /// </summary>
    protected override void OnEnabled()
    {
        CreateWorldModel();
    }

    /// <summary>
    /// Когда выключается — уничтожаем модели.
    /// </summary>
    protected override void OnDisabled()
    {
        DestroyWorldModel();
        DestroyViewModel();
    }

    // --- Обновление ---

    protected override void OnUpdate()
    {
        var player = Owner;
        var controller = player?.Controller;
        if ( controller is null ) return;

        // Рисуем HUD только для локального игрока
        if ( player.IsLocalPlayer )
        {
            if ( Scene.Camera is null ) return;
            var hud = Scene.Camera.Hud;
            var aimPos = Screen.Size * 0.5f; // Центр экрана
            DrawHud( hud, aimPos );
        }
    }

    /// <summary>
    /// Рисует HUD (прицел, информация). Переопределяется в наследниках.
    /// </summary>
    public virtual void DrawHud( HudPainter painter, Vector2 crosshair )
    {
        // По умолчанию ничего не рисует
    }

    // --- Виртуальные методы для наследников ---

    /// <summary>
    /// Вызывается когда предмет добавлен в инвентарь.
    /// </summary>
    public virtual void OnAdded( Player player ) { }

    /// <summary>
    /// Вызывается каждый кадр, пока предмет активен.
    /// </summary>
    public virtual void OnFrameUpdate( Player player ) { }

    /// <summary>
    /// Вызывается каждый кадр от лица владельца.
    /// Здесь обрабатывается ввод (стрельба, перезарядка).
    /// </summary>
    public virtual void OnPlayerUpdate( Player player ) { }

    /// <summary>
    /// Вызывается при смерти владельца.
    /// </summary>
    public virtual void OnPlayerDeath( IPlayerEvent.DiedParams args ) { }

    /// <summary>
    /// Можно ли переключиться на это оружие?
    /// </summary>
    public virtual bool CanSwitch() => true;

    /// <summary>
    /// Стоит ли избегать это оружие при автопереключении?
    /// </summary>
    public virtual bool ShouldAvoid => false;

    /// <summary>
    /// Используется ли сейчас (стреляет, перезаряжает)?
    /// </summary>
    public virtual bool IsInUse() => false;
}
```

### Ключевые паттерны:

#### `GetComponentInParent<T>()`

Это метод, который ищет компонент **вверх** по иерархии объектов:

```
GameObject "Игрок"
├── Player                    ← GetComponentInParent<Player>() найдёт его
├── PlayerInventory
└── "M4A1"                    ← мы здесь
    └── M4a1Weapon : BaseCarryable
```

#### `virtual` и `override`

`virtual` означает «этот метод можно переопределить в наследнике». Это позволяет каждому оружию иметь своё поведение:

```csharp
// В BaseCarryable:
public virtual void DrawHud(...) { } // ничего

// В BaseBulletWeapon:
public override void DrawHud(...) { /* рисует прицел */ }

// В SniperWeapon:
public override void DrawHud(...) { /* рисует прицел снайпера */ }
```

## Шаг 6.2: BaseWeapon — оружие с патронами

Создай файл `Code/Game/Weapon/BaseWeapon/BaseWeapon.cs`:

```csharp
/// <summary>
/// Оружие с патронами, перезарядкой и стрельбой.
/// Наследуется от BaseCarryable, добавляя механику боеприпасов.
/// </summary>
public partial class BaseWeapon : BaseCarryable, IPlayerControllable
{
    // --- Настройки стрельбы ---

    /// <summary>
    /// Интервал между выстрелами (в секундах). 
    /// 0.1 = 10 выстрелов/сек, 0.5 = 2 выстрела/сек
    /// </summary>
    [Property] public float FireRate { get; set; } = 0.1f;

    /// <summary>
    /// Автоматическая стрельба? Если true — стреляет пока зажата кнопка.
    /// </summary>
    [Property] public bool Automatic { get; set; } = true;

    // --- Боеприпасы ---
    
    [Property] public bool UsesAmmo { get; set; } = true;
    [Property] public bool UsesClips { get; set; } = true;
    [Property] public int ClipSize { get; set; } = 30;
    [Property] public int StartingAmmo { get; set; } = 30;
    [Property] public int MaxReserveAmmo { get; set; } = 90;

    /// <summary>
    /// Текущие патроны в магазине.
    /// </summary>
    [Sync( SyncFlags.FromHost )] 
    public int ClipContents { get; set; }

    /// <summary>
    /// Запас патронов (не в магазине).
    /// </summary>
    [Sync( SyncFlags.FromHost )] 
    public int ReserveAmmo { get; set; }

    // --- Таймеры ---
    
    private TimeSince _lastFireTime;
    [Sync] public bool IsReloading { get; private set; }
    [Property] public float ReloadTime { get; set; } = 2.4f;

    // --- Инициализация ---

    protected override void OnEnabled()
    {
        base.OnEnabled();
        
        if ( ClipContents == 0 && !IsReloading )
        {
            ClipContents = ClipSize;
            ReserveAmmo = StartingAmmo;
        }
    }

    // --- Стрельба ---

    public override void OnPlayerUpdate( Player player )
    {
        // Перезарядка
        if ( Input.Pressed( "reload" ) )
        {
            TryReload();
            return;
        }

        // Стрельба
        bool wantsToShoot = Automatic ? Input.Down( "attack1" ) : Input.Pressed( "attack1" );

        if ( wantsToShoot && CanShoot() )
        {
            Shoot();
        }
    }

    /// <summary>
    /// Можно ли стрелять прямо сейчас?
    /// </summary>
    protected virtual bool CanShoot()
    {
        if ( IsReloading ) return false;
        if ( _lastFireTime < FireRate ) return false;
        if ( UsesAmmo && ClipContents <= 0 ) return false;
        return true;
    }

    /// <summary>
    /// Выстрел. Переопределяется в наследниках.
    /// </summary>
    protected virtual void Shoot()
    {
        _lastFireTime = 0;
        
        if ( UsesAmmo )
            ClipContents--;

        // Наследники (BaseBulletWeapon) переопределяют этот метод
    }

    /// <summary>
    /// Проверяет, есть ли патроны.
    /// </summary>
    public bool HasAmmo()
    {
        if ( !UsesAmmo ) return true;
        return ClipContents > 0;
    }

    /// <summary>
    /// Можно ли перезарядиться?
    /// </summary>
    public bool CanReload()
    {
        if ( !UsesAmmo || !UsesClips ) return false;
        if ( IsReloading ) return false;
        if ( ClipContents >= ClipSize ) return false;
        if ( ReserveAmmo <= 0 ) return false;
        return true;
    }

    /// <summary>
    /// Начать перезарядку.
    /// </summary>
    public void TryReload()
    {
        if ( !CanReload() ) return;
        StartReload();
    }

    private async void StartReload()
    {
        IsReloading = true;

        // Ждём время перезарядки
        await Task.DelayRealtimeSeconds( ReloadTime );

        if ( !this.IsValid() ) return;

        FinishReload();
    }

    private void FinishReload()
    {
        IsReloading = false;

        if ( !UsesAmmo ) return;

        var needed = ClipSize - ClipContents;
        var available = Math.Min( needed, ReserveAmmo );
        
        ClipContents += available;
        ReserveAmmo -= available;
    }

    /// <summary>
    /// Добавить патроны в запас.
    /// </summary>
    public void AddReserveAmmo( int amount )
    {
        ReserveAmmo = Math.Min( ReserveAmmo + amount, MaxReserveAmmo );
    }

    public override bool IsInUse()
    {
        return IsReloading || Input.Down( "attack1" );
    }
}
```

### Как работает перезарядка:

```
Клавиша R нажата
    ↓
CanReload()? → ClipContents < ClipSize? → ReserveAmmo > 0?
    ↓ да
StartReload() → IsReloading = true
    ↓
Ждём ReloadTime секунд (async/await)
    ↓
FinishReload() → Перемещаем патроны из ReserveAmmo в ClipContents
    ↓
IsReloading = false → Можно стрелять!
```

## Шаг 6.3: BaseBulletWeapon — стрелковое оружие

Создай файл `Code/Game/Weapon/BaseBulletWeapon/BaseBulletWeapon.cs`:

```csharp
/// <summary>
/// Стрелковое оружие, использующее рейкаст (мгновенные лучи).
/// Hitscan — пуля летит мгновенно от дула до цели.
/// </summary>
public partial class BaseBulletWeapon : BaseWeapon
{
    // --- Настройки урона ---
    
    [Property] public float BaseDamage { get; set; } = 10f;
    [Property] public float MaxRange { get; set; } = 4096f;
    
    /// <summary>
    /// Разброс пуль. 0 = идеально точно, 0.05 = небольшой разброс.
    /// </summary>
    [Property] public float Spread { get; set; } = 0.01f;

    /// <summary>
    /// Количество пуль за выстрел (для дробовика > 1).
    /// </summary>
    [Property] public int BulletsPerShot { get; set; } = 1;

    /// <summary>
    /// Сила отдачи.
    /// </summary>
    [Property] public float RecoilForce { get; set; } = 1f;

    // --- Стрельба ---

    protected override void Shoot()
    {
        base.Shoot(); // Уменьшает патроны, обновляет таймер

        // Для каждой пули (у дробовика их несколько)
        for ( int i = 0; i < BulletsPerShot; i++ )
        {
            ShootBullet();
        }

        // Звук и эффекты
        ShootEffects();
    }

    /// <summary>
    /// Один выстрел — рейкаст от глаз игрока вперёд.
    /// </summary>
    [Rpc.Host]
    protected virtual void ShootBullet()
    {
        var player = Owner;
        if ( !player.IsValid() ) return;

        var eyePos = player.EyeTransform.Position;
        var eyeDir = player.EyeTransform.Forward;

        // Добавляем разброс
        eyeDir += Vector3.Random * Spread;
        eyeDir = eyeDir.Normal;

        // Рейкаст — пускаем луч от глаз вперёд
        var tr = Scene.Trace
            .Ray( eyePos, eyePos + eyeDir * MaxRange )
            .IgnoreGameObjectHierarchy( player.GameObject )
            .WithoutTags( "trigger" )
            .Run();

        // Если попали во что-то
        if ( tr.Hit )
        {
            // Находим IDamageable на цели
            var damageable = tr.GameObject?.GetComponent<Component.IDamageable>();
            if ( damageable != null )
            {
                var damage = BaseDamage;
                var info = new DamageInfo( damage, player.GameObject, GameObject );

                // Проверяем попадание в голову
                if ( tr.Hitbox?.Tags?.Has( "head" ) ?? false )
                    info.Tags.Add( DamageTags.Headshot );

                damageable.OnDamage( in info );
            }
        }

        // Создаём визуальный след пули
        CreateTracer( eyePos, tr.EndPosition );
    }

    /// <summary>
    /// Эффекты выстрела (звук, вспышка, гильза).
    /// Выполняется у ВСЕХ клиентов.
    /// </summary>
    [Rpc.Broadcast]
    protected virtual void ShootEffects()
    {
        // Звук выстрела — наследники указывают свой звук
    }

    /// <summary>
    /// Рисует след пули.
    /// </summary>
    [Rpc.Broadcast]
    protected void CreateTracer( Vector3 from, Vector3 to )
    {
        // Создаём частицу-трассер от дула до точки попадания
    }

    // --- HUD ---

    public override void DrawHud( HudPainter painter, Vector2 crosshair )
    {
        // Рисуем прицел
        float size = 4f;
        float gap = 8f;

        // Четыре линии прицела
        painter.DrawLine( 
            crosshair + new Vector2( -gap - size, 0 ), 
            crosshair + new Vector2( -gap, 0 ), 
            Color.White, 2f );
        painter.DrawLine( 
            crosshair + new Vector2( gap, 0 ), 
            crosshair + new Vector2( gap + size, 0 ), 
            Color.White, 2f );
        painter.DrawLine( 
            crosshair + new Vector2( 0, -gap - size ), 
            crosshair + new Vector2( 0, -gap ), 
            Color.White, 2f );
        painter.DrawLine( 
            crosshair + new Vector2( 0, gap ), 
            crosshair + new Vector2( 0, gap + size ), 
            Color.White, 2f );
    }
}
```

### Как работает рейкаст (Hitscan):

```
Глаз игрока ──────── [луч] ──────────→ Стена
     ↑                                    ↑
  eyePos                             tr.EndPosition
                      
Если на пути есть враг:
     
Глаз ──── [луч] ──→ ВРАГ ──→ (луч останавливается)
                      ↑
                  tr.Hit = true
                  tr.GameObject = враг
```

**Scene.Trace.Ray()** — функция движка, которая пускает невидимый луч из точки A в точку B. Она возвращает:
- `Hit` — попал ли луч во что-то
- `GameObject` — в какой объект попал
- `HitPosition` — точка попадания
- `Hitbox` — в какой хитбокс попал (голова, тело...)
- `Normal` — нормаль поверхности (для эффектов)

### Почему ShootBullet() помечен [Rpc.Host]?

Расчёт попадания и урона должен происходить **только на хосте** (сервере). Иначе клиент мог бы «стрелять» мимо, но всё равно наносить урон.

```
Клиент нажимает мышь:
1. Shoot() → ShootBullet() [Rpc.Host] → отправляется на хост
2. Хост считает рейкаст → наносит урон
3. ShootEffects() [Rpc.Broadcast] → все видят вспышку и слышат звук
```
