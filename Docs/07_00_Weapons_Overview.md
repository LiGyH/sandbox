## Общий паттерн всех «коротких» оружий (07.01–07.04, 07.07)

Файлы 07.01–07.04 и 07.07 — это **один и тот же скелет**:

```csharp
public class XxxWeapon : IronSightsWeapon   // или MeleeWeapon для монтировки
{
    [Property] public float PrimaryFireRate { get; set; } = 0.2f;

    protected override float GetPrimaryFireRate() => PrimaryFireRate;

    protected override bool WantsPrimaryAttack()
    {
        return Input.Pressed( "attack1" );   // или Input.Down для автоматов
    }

    public override void PrimaryAttack()
    {
        ShootBullet( PrimaryFireRate, GetBullet() );
    }

    public override void DrawCrosshair( HudPainter hud, Vector2 center )
    {
        // обычно: точка 5px чёрная + 3px цветная (зелёная/красная)
    }
}
```

Различия между пистолетами / автоматами сводятся к трём вещам:
1. **`Input.Pressed` vs `Input.Down`** — одиночные или очередь.
2. **Значение `PrimaryFireRate`** — как быстро.
3. **Префаб оружия** в ассетах — модель, звуки, урон, магазин.

Остальное — базовый `IronSightsWeapon` (прицеливание через мушку), `BaseBulletWeapon` (выстрел, трассировка, урон), `BaseWeapon` (ввод, перезарядка).

> Отсюда вывод: если хочешь **добавить своё огнестрельное оружие**, тебе хватит создать префаб + класс на 20 строк по этому шаблону. Не трогай базовые классы.

## Уникальные оружия — на что обращать внимание

### 07.05 Shotgun — разброс и помпа
- `NumShots` — сколько пуль выпустить за один выстрел.
- `Spread` — насколько пули расходятся (делает оружие ближнего боя).
- Разделение «выстрел» и «передёрнуть затвор» по таймерам.

### 07.06 Sniper — зум-прицел
- `Zoom()` / `Unzoom()` — изменение `Camera.FieldOfView` для эффекта снайперского зума.
- Отдельный рисунок прицела в зуме (крест + шкала), обычного — в бедре.
- Задержка после каждого выстрела перед следующим (перезарядка затвора).

### 07.08 HandGrenade — физика и сеть
- `ThrowGrenade()` создаёт `GameObject` с `Rigidbody` и метает его с `ApplyImpulse`.
- Таймер детонации через `async Task.Delay` — обрати внимание, как это работает в `Sandbox`-корутинах.
- При взрыве — сферический поиск жертв (`Scene.Trace.Sphere`), раздача урона, партиклы, звук.
- Физика летящей гранаты синхронизируется через `GameObject.NetworkMode = Object`.

### 07.09 CameraWeapon — скриншоты
- `Screen.CaptureImage()` — движок s&box делает снимок экрана без HUD.
- Сохранение: `FileSystem.Data.WriteStream(...)` — см. [Facepunch/sbox-docs FileSystem](https://github.com/Facepunch/sbox-docs/).
- Вспышка затвора реализована через `HudPainter.DrawRect` с анимацией прозрачности.
