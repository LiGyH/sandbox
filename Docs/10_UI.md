# 10 — Пользовательский интерфейс (UI) 🖥️

## Обзор

UI построен на Razor-компонентах s&box (аналог Blazor). Файлы `.razor` содержат HTML-подобную разметку + C# логику. Стили — через `.scss` файлы рядом с `.razor`.

## 10.02 — Спавн-меню (SpawnMenu)

Открывается по **Q**. Состоит из вкладок:

### Props (пропы)
- Поиск по имени
- Облачный браузер (Cloud)
- Избранное
- Иконки с `thumb:path`
- ЛКМ по иконке → `GameManager.Spawn("prop:path")`
- Перетаскивание в хотбар → `SpawnerWeapon`

### Entities (сущности)
- Локальные + облачные
- Категории (Light, Physics, NPC...)
- Фильтр developer entities (`ServerSettings.ShowDeveloperEntities`)

### Spawnlists (спавнлисты)
- `SpawnlistData` + `SpawnlistCollection` — управление коллекциями
- Создание, редактирование, Workshop

### Dupes (дупликаты)
- Локальные (из `Storage`)
- Workshop (из Steam Workshop)
- ЛКМ → выбрать дупликат для Duplicator tool

### Tools (инструменты)
- Список всех `ToolMode` из `TypeLibrary`
- ЛКМ → переключить Toolgun на этот режим

### Mounts (маунты)
- Содержимое из установленных игр
- Установка/удаление маунтов

## 10.03 — Инвентарь (Inventory)

Хотбар внизу экрана:
- `Inventory.razor` — контейнер слотов
- `InventorySlot.razor` — один слот

- Колесо мыши → следующий/предыдущий
- Drag-and-drop между слотами
- `HotbarPresetsButton.razor` — пресеты loadout'а

## 10.07 — Контекстное меню (Context Menu)

Открывается по **C** при наведении на объект:
- `Inspector.razor` — показывает свойства объекта
- `ComponentHandle.cs` — редактор компонента
- `InspectorEditor.cs` — кастомные редакторы
- Изменения отправляются через `GameManager.ChangeProperty()`

Система drag-and-drop для перетаскивания из спавн-меню в хотбар.

## Razor-компоненты: как это работает

```razor
@using Sandbox.UI

<root class="my-panel">
    <div class="title">@Title</div>
    
    @if ( IsVisible )
    {
        <div class="content">
            @foreach ( var item in Items )
            {
                <SpawnMenuIcon Item=@item />
            }
        </div>
    }
</root>

@code {
    public string Title { get; set; }
    public bool IsVisible { get; set; }
    public List<Item> Items { get; set; }
}
```

- HTML-подобная разметка с C# вставками (`@expression`)
- `@code {}` блок — свойства и методы
- Стили через `.scss` файлы
- Автоматическое обновление при изменении свойств
