# Этап 13: Интерфейс (UI)

## Цель этого этапа

Создать пользовательский интерфейс: меню спавна, HUD (здоровье, патроны), уведомления, килл-фид. 

## Как работает UI в s&box

s&box использует систему UI, основанную на **Razor-компонентах** (похожа на HTML + C#).

```
UI файлы:
├── .razor      — разметка (как HTML)
├── .razor.cs   — логика (C#)
└── .razor.scss — стили (как CSS)
```

### Где рисуется UI:

UI рисуется в **системной сцене** (`system.scene`). Она загружается поверх основной сцены и содержит:

```
system.scene
├── Screen Panel       ← Корневой элемент UI
│   ├── Hud            ← Здоровье, патроны, прицел
│   ├── SpawnMenu      ← Меню спавна (Q)
│   ├── Chat           ← Чат
│   ├── Feed           ← Килл-фид
│   ├── Notices        ← Уведомления
│   └── ContextMenu    ← Контекстное меню (правый клик)
├── Camera Component   ← Камера
└── HUD Component      ← Рисовалка 2D
```

## Шаг 13.1: Системная сцена

Системная сцена (`scenes/system.scene`) — это JSON-файл, который описывает UI-дерево. Она указана в `.sbproj`:

```json
"SystemScene": "scenes/system.scene"
```

Она загружается **параллельно** с основной сценой и **поверх** неё.

## Шаг 13.2: HUD — Здоровье и патроны

HUD рисуется через `HudPainter` — встроенный класс для рисования 2D-графики поверх 3D-сцены.

Создай файл `Code/Utility/HUD.cs`:

```csharp
/// <summary>
/// Система HUD. Рисует здоровье, броню, патроны.
/// Использует CameraComponent.Hud для рисования прямо на экране.
/// </summary>
public class HudSystem : GameObjectSystem<HudSystem>
{
    public HudSystem( Scene scene ) : base( scene )
    {
        // Подписываемся на событие рисования HUD
        Listen( Stage.FinishUpdate, 100, DrawHud, "DrawHud" );
    }

    void DrawHud()
    {
        var camera = Scene.Camera;
        if ( camera is null ) return;

        var hud = camera.Hud;
        var player = Player.FindLocalPlayer();
        if ( !player.IsValid() ) return;
        if ( player.WantsHideHud ) return;

        DrawHealthBar( hud, player );
        DrawAmmo( hud, player );
    }

    void DrawHealthBar( HudPainter hud, Player player )
    {
        var screenSize = Screen.Size;
        var pos = new Vector2( 20, screenSize.y - 60 );

        // Фон
        hud.DrawRect( 
            new Rect( pos.x, pos.y, 200, 30 ), 
            Color.Black.WithAlpha( 0.5f ) );

        // Полоска здоровья
        var healthFraction = player.Health / player.MaxHealth;
        var healthColor = healthFraction > 0.3f ? Color.Green : Color.Red;
        
        hud.DrawRect( 
            new Rect( pos.x + 2, pos.y + 2, 196 * healthFraction, 26 ), 
            healthColor );

        // Текст
        hud.DrawText( 
            new Rect( pos.x, pos.y, 200, 30 ), 
            $"{(int)player.Health}", 
            Color.White, 
            "Consolas", 
            16 );

        // Броня (если есть)
        if ( player.Armour > 0 )
        {
            pos.y -= 35;
            hud.DrawRect( 
                new Rect( pos.x, pos.y, 200, 25 ), 
                Color.Black.WithAlpha( 0.5f ) );
            
            var armourFraction = player.Armour / player.MaxArmour;
            hud.DrawRect( 
                new Rect( pos.x + 2, pos.y + 2, 196 * armourFraction, 21 ), 
                Color.Cyan );
        }
    }

    void DrawAmmo( HudPainter hud, Player player )
    {
        var inventory = player.GetComponent<PlayerInventory>();
        if ( !inventory.IsValid() ) return;

        var weapon = inventory.ActiveWeapon as BaseWeapon;
        if ( weapon == null || !weapon.UsesAmmo ) return;

        var screenSize = Screen.Size;
        var pos = new Vector2( screenSize.x - 220, screenSize.y - 60 );

        // Патроны в магазине / запас
        var text = weapon.UsesClips
            ? $"{weapon.ClipContents} / {weapon.ReserveAmmo}"
            : $"{weapon.ReserveAmmo}";

        hud.DrawText( 
            new Rect( pos.x, pos.y, 200, 30 ), 
            text, 
            Color.White, 
            "Consolas", 
            20 );

        if ( weapon.IsReloading )
        {
            hud.DrawText( 
                new Rect( pos.x, pos.y - 25, 200, 20 ), 
                "RELOADING...", 
                Color.Yellow, 
                "Consolas", 
                12 );
        }
    }
}
```

## Шаг 13.3: Меню спавна

Меню спавна открывается клавишей **Q**. Оно показывает:
- Список пропов (моделей)
- Список сущностей
- Список инструментов
- Дупликаты

Создай файл `Code/UI/ISpawnMenuTab.cs`:

```csharp
/// <summary>
/// Интерфейс для вкладок меню спавна.
/// Каждая вкладка (Props, Entities, Tools) реализует этот интерфейс.
/// </summary>
public interface ISpawnMenuTab
{
    string TabTitle { get; }
    string TabIcon { get; }
    int TabOrder => 100;
}
```

В полной версии меню спавна реализовано через Razor-компоненты:

```
SpawnMenu (BaseSpawnMenu.razor)
├── PropsPage     ← Вкладка "Пропы"
├── EntityPage    ← Вкладка "Сущности"  
├── MountsPage    ← Вкладка "Монтирования"
├── DupesPage     ← Вкладка "Дупликаты"
└── SpawnlistsPage← Вкладка "Спавн-листы"
```

### Razor-компонент — пример:

```razor
@* Это упрощённый пример Razor-компонента *@
@inherits PanelComponent

<root class="spawn-menu">
    <div class="tabs">
        <button onclick="@(() => CurrentTab = "props")">Props</button>
        <button onclick="@(() => CurrentTab = "ents")">Entities</button>
    </div>
    
    <div class="content">
        @if ( CurrentTab == "props" )
        {
            @foreach ( var prop in PropList )
            {
                <div class="prop-item" onclick="@(() => SpawnProp(prop))">
                    <img src="@prop.Icon" />
                    <span>@prop.Name</span>
                </div>
            }
        }
    </div>
</root>
```

## Шаг 13.4: Килл-фид (Feed)

Килл-фид показывает убийства в правом верхнем углу.

Создай файл `Code/UI/IKillIcon.cs`:

```csharp
/// <summary>
/// Интерфейс для иконки в килл-фиде.
/// Оружие реализует его, чтобы показывать свою иконку.
/// </summary>
public interface IKillIcon
{
    string DisplayName { get; }
    Texture DisplayIcon { get; }
}
```

Килл-фид получает данные через `Scene.RunEvent<Feed>()`:

```csharp
// В GameManager.OnDeath():
Scene.RunEvent<Feed>( x => x.NotifyKill( 
    victimName, attackerName, steamId, damageTags, 
    attackerTags, "", weaponIcon ) );
```

## Шаг 13.6: Контекстное меню (Context Menu)

Контекстное меню открывается при **клике по объекту** в режиме инспектора. Позволяет:
- Менять свойства объекта (цвет, размер, масса)
- Удалять объект
- Копировать и вставлять

В полной версии используется `InspectorEditor.cs` и `ComponentHandle.cs`.
