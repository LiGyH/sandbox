# 26.09 — Color и цветовые пространства

## Что мы делаем?

Цвет используется буквально везде: HUD, эффекты, свет, шейдеры, килл-фид. На первый взгляд `Color` — это «четыре числа RGBA», но у него есть нюансы: разные пространства, прозрачность, конвертации, стандартные палитры.

## Базовое создание

```csharp
var red   = new Color( 1f, 0f, 0f );        // R, G, B (0..1)
var green = new Color( 0f, 1f, 0f, 0.5f );  // с альфой 50%
var blue  = Color.Blue;                     // встроенная константа
```

Каналы — **`float` от 0 до 1**, не `byte` 0..255. Это упрощает математику (умножение, лерп) и даёт HDR-цвета.

## HDR-цвета (`> 1.0`)

В s&box цвет может быть **ярче белого**: `new Color(2, 2, 2)` — это «солнце через окно». Эти значения важны для:

- источников света (свет-эмиттер с интенсивностью 5);
- post-process bloom;
- частиц «вспышка от взрыва».

Не «зажимай» цвет в `[0..1]` без необходимости — потеряешь HDR.

## Альфа-канал

`Color.A = 0` — полностью прозрачно, `1` — непрозрачно. Удобный «смыслошагающий» метод:

```csharp
panel.BackgroundColor = Color.Black.WithAlpha( 0.5f );
```

`WithAlpha` возвращает **новый** цвет (как все методы `With*`).

## Цветовые пространства

| Пространство | Как создать | Для чего |
|---|---|---|
| **RGB** | `new Color( r, g, b )` | базовое, прямое для шейдеров и UI |
| **HSV** | `Color.FromHsv( h, s, v )` | удобно крутить **оттенок** по кругу |
| **HEX** | `Color.Parse( "#FF8800" )` | копирование из дизайн-инструментов |
| **Bytes** | `new Color32( 255, 128, 0 )` | если у тебя данные пришли в `byte` |

```csharp
// Радуга по тику:
float hue = (Time.Now * 0.1f) % 1f;
var rainbow = Color.FromHsv( hue * 360f, 1f, 1f );

// Из строки:
var brand = Color.Parse( "#3498DB" );
```

## Готовые константы

`Color.White`, `Color.Black`, `Color.Red`, `Color.Green`, `Color.Blue`, `Color.Yellow`, `Color.Orange`, `Color.Cyan`, `Color.Magenta`, `Color.Transparent`. Используй их вместо магических чисел.

## Операции

```csharp
var mid = Color.Lerp( a, b, 0.5f );        // микс
var darker = c * 0.5f;                     // умножение на скаляр
var summed = a + b;                        // сложение каналов
var inverted = new Color( 1 - c.R, 1 - c.G, 1 - c.B, c.A );
```

> ⚠️ Сумма цветов может вылезти за `1.0` — это нормально для HDR, но в UI может выглядеть «выгоревшим».

## Линейное vs gamma-пространство

Технически в s&box цвета хранятся в **линейном** пространстве (правильно для физически корректного освещения), а на экран выводятся через **gamma-коррекцию (sRGB)**. Тебе не нужно делать это вручную — движок справится.

Подвох случается, когда ты берёшь HEX-цвет из Photoshop (он в sRGB) и пихаешь в шейдер как линейный — получится **темнее**. В таких местах используй `Color.SrgbToLinear()` / `Color.LinearToSrgb()`.

## Пример: индикатор здоровья

```csharp
public Color HealthColor( float fraction )
{
    // 1.0 — зелёный, 0.5 — жёлтый, 0.0 — красный
    if ( fraction > 0.5f )
        return Color.Lerp( Color.Yellow, Color.Green, (fraction - 0.5f) * 2f );
    return Color.Lerp( Color.Red, Color.Yellow, fraction * 2f );
}
```

…или используй `Gradient` из [26.08](26_08_Curve_Easing.md) — будет проще и настраиваемо в инспекторе.

## Цвет в Razor (CSS)

В стилях SCSS цвет можно писать:

```scss
.health-bar {
    background-color: rgba(255, 64, 64, 0.8);
    color: #FF8800;
}
```

В коде Razor — через интерполяцию:

```csharp
<div style="background-color: @($"rgba(255, 0, 0, {alpha})")">
```

Но обычно проще: `panel.Style.BackgroundColor = Color.Red.WithAlpha( 0.5f );`.

## Что важно запомнить

- `Color` — четыре `float` от 0 до 1 (HDR — больше).
- `WithAlpha`, `Lerp`, `FromHsv`, `Parse("#...")` — основные удобства.
- HSV годится для радуги/оттенков, HEX — для дизайнерских цветов.
- Различай **линейное** и **sRGB**, если работаешь с шейдерами.
- В UI лучше не складывать цвета руками — пользуйся `Color.Lerp` или `Gradient`.
