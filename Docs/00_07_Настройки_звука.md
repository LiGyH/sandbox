# 00.07 — Настройки звука (Mixer)

## Что мы делаем?

Настраиваем **аудио-микшер** — систему, которая управляет громкостью разных типов звуков.

## Зачем нужен микшер?

Без микшера все звуки играют на одной громкости. Но в игре нужно:
- Музыка играет тише чем выстрелы
- Звуки UI (кнопки меню) не должны быть пространственными (не зависят от направления)
- Выстрелы в 3D мире должны быть тише вдали и громче вблизи

## Структура микшера (дерево)

```
Master (Volume: 1.0)
├── Music (Volume: 1.0, без пространственного звука)
├── Game (Volume: 1.0, по умолчанию)
│   └── Weapons (Volume: 0.25, пространственный)
│       ├── FirstPerson (от первого лица)
│       └── ThirdPerson (от третьего лица)
└── UI (Volume: 0.75, без пространственного звука)
```

## Что означают параметры?

| Параметр | Что делает |
|----------|------------|
| `Volume` | Громкость (0 = тишина, 1 = максимум) |
| `Spacializing` | Пространственный звук (1 = звук зависит от позиции, 0 = нет) |
| `MaxVoices` | Максимум одновременных звуков в канале |
| `DistanceAttenuation` | Затухание с расстоянием (1 = дальше = тише) |
| `Occlusion` | Преграды заглушают звук (стена между тобой и источником) |
| `AirAbsorption` | Воздух поглощает высокие частоты на расстоянии |
| `IsDefault` | Звуки без указанного канала попадут сюда |

> **Как это работает внутри движка:**  
> Класс `MixerSettings` загружается системой аудио.  
> Каждый звук назначается на **канал** микшера (Game, Music, UI, Weapons).  
> Финальная громкость = `Master.Volume × Parent.Volume × Channel.Volume × Source.Volume`.
>
> Например, выстрел оружия: `1.0 × 1.0 × 0.25 × 1.0 = 0.25` (25% громкости).
> Это сделано чтобы выстрелы не оглушали — они очень громкие в реальности.
