# 07.09 — Монтировка (CrowbarWeapon) 🔧

## Что мы делаем?

Создаём конкретный класс оружия `CrowbarWeapon` — оружие ближнего боя (монтировка), наследующее `MeleeWeapon`.

## Зачем это нужно?

- **Простейшее оружие ближнего боя**: класс полностью полагается на логику базового `MeleeWeapon` без каких-либо переопределений.
- **Точка расширения**: отдельный класс позволяет в будущем добавить уникальное поведение (например, особые анимации, звуки, урон) без изменения базового класса.
- **Уникальный тип**: движок идентифицирует оружие по типу класса, что позволяет системе инвентаря, статистике и логике различать монтировку от других мили-оружий.

## Как это работает внутри движка?

| Элемент | Описание |
|---|---|
| `MeleeWeapon` | Базовый класс, содержащий всю логику ближнего боя: трассировка удара, урон, анимации, задержка между атаками. |
| Пустое тело класса | Всё поведение унаследовано. Параметры (урон, дальность, скорость) задаются через инспектор на базовом классе. |

## Создай файл

**Путь:** `Code/Weapons/CrowbarWeapon.cs`

```csharp
public class CrowbarWeapon : MeleeWeapon
{
}
```

## Проверка

1. Возьмите монтировку и нажмите ЛКМ — должна проиграться анимация удара.
2. Ударьте по объекту — он должен получить урон и/или отлететь.
3. Убедитесь, что между ударами есть задержка (нельзя спамить мгновенно).
4. Проверьте, что монтировка корректно отображается в инвентаре с правильной иконкой и названием.
