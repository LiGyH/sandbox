# Этап 05_01 — Стили Theme и Hud

## Что это такое?

Представьте, что вы оформляете комнату: выбираете цвет стен, шрифт на табличках, размеры мебели. **Theme.scss** и **Hud.scss** — это «дизайн-проект» для всего игрового интерфейса. Они задают цвета, шрифты и общие правила оформления, чтобы все элементы на экране выглядели единообразно.

**SCSS** — это расширенный CSS (язык стилей для веб-страниц). В s&box интерфейс строится похоже на веб-страницы, поэтому стили пишутся в формате SCSS.

---

## Файл `Theme.scss`

Этот файл — «палитра художника». Здесь объявляются переменные (начинаются с `$`), которые потом используются во всех остальных файлах стилей.

### Полный исходный код

**Путь:** `Code/UI/Theme.scss`

```scss
$deadzone-x: 2vw;
$deadzone-y: 2vw;

$color-accent: #0094ff;
$color-text: #fafaff;

$title-font: "Poppins";
$body-font: "Inter";
$subtitle-font: "Poppins";

@import "/UI/Menu.scss";
@import "/UI/Hud.scss";
```

### Разбор кода

| Строка | Что делает |
|--------|-----------|
| `$deadzone-x: 2vw;` | Переменная — отступ от края экрана по горизонтали. `2vw` означает 2% ширины экрана |
| `$deadzone-y: 2vw;` | Отступ от края экрана по вертикали |
| `$color-accent: #0094ff;` | Основной акцентный цвет (голубой). `#0094ff` — это цвет в формате HEX |
| `$color-text: #fafaff;` | Цвет текста — почти белый |
| `$title-font: "Poppins";` | Шрифт для заголовков |
| `$body-font: "Inter";` | Шрифт для основного текста |
| `$subtitle-font: "Poppins";` | Шрифт для подзаголовков |
| `@import "/UI/Menu.scss";` | Подключает файл стилей меню |
| `@import "/UI/Hud.scss";` | Подключает файл стилей HUD |

> **Для новичков:** Переменные (строки с `$`) — это как «имена» для значений. Вместо того чтобы писать `#0094ff` в каждом файле, мы пишем `$color-accent`. Если захотим поменять цвет — изменим только одну строку.

## Файл `Hud.scss`

Этот файл описывает общий стиль для панелей HUD (heads-up display — элементы на экране во время игры).

### Полный исходный код

**Путь:** `Code/UI/Hud.scss`

```scss
$hud-element-bg: #0006;
$hud-text: #fafaff;

$hud-radius: 8px;
$hud-radius-sm: 4px;
$hud-icon-size: 30px;

.hud-panel
{
	flex-direction: row;
	align-items: center;
	gap: 16px;
	padding: 22px 22px;
	background-color: $hud-element-bg;
	border-radius: $hud-radius;
	font-family: $title-font;
	pointer-events: none;

	Image, .icon
	{
		width: $hud-icon-size;
		height: 36px;
		background-size: contain;
		background-repeat: no-repeat;
		background-position: center;
		flex-shrink: 0;
		background-tint: $color-accent;
	}

	.value
	{
		font-size: 29px;
		font-weight: 700;
		color: $hud-text;
	}

	.alternate
	{
		font-size: 29px;
		font-weight: 700;
		color: #fffa;
	}
}
```

### Разбор кода

**Переменные:**

| Переменная | Значение | Что это |
|-----------|----------|---------|
| `$hud-element-bg` | `#0006` | Полупрозрачный чёрный фон. `#0006` — это сокращение: чёрный цвет с прозрачностью ~40% |
| `$hud-text` | `#fafaff` | Цвет текста HUD (почти белый) |
| `$hud-radius` | `8px` | Скругление углов панелей |
| `$hud-radius-sm` | `4px` | Маленькое скругление углов |
| `$hud-icon-size` | `30px` | Размер иконок |

**Класс `.hud-panel`:**

Это базовый стиль для всех панелей на экране (здоровье, патроны и т.д.):

- `flex-direction: row;` — элементы внутри располагаются в строку (горизонтально)
- `align-items: center;` — выравнивание по центру по вертикали
- `gap: 16px;` — расстояние между элементами внутри панели
- `padding: 22px 22px;` — внутренний отступ от краёв панели
- `background-color: $hud-element-bg;` — полупрозрачный чёрный фон
- `border-radius: $hud-radius;` — скруглённые углы
- `pointer-events: none;` — панель не перехватывает клики мыши (чтобы не мешать игре)

**Вложенные стили:**

- `Image, .icon` — стиль для иконок: фиксированный размер, иконка вписывается в блок, подкрашивается акцентным цветом
- `.value` — крупный жирный белый текст для основных значений (например, «100» здоровья)
- `.alternate` — такой же текст, но чуть прозрачнее (`#fffa`) — для второстепенных значений (например, запас патронов)

---

## Как это работает вместе

```
Theme.scss (главный файл)
├── Объявляет переменные: цвета, шрифты, отступы
├── @import Menu.scss
└── @import Hud.scss
    └── Использует переменные из Theme.scss
        └── Определяет .hud-panel — базовый стиль для всех HUD-элементов
```

Когда любой компонент (здоровье, патроны, чат) пишет `@import "/UI/Theme.scss"` — он получает доступ ко всем переменным и стилю `.hud-panel`.
