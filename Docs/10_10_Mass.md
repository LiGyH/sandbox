## Что мы делаем?

Создаём файл `Mass.cs` — инструмент для **изменения массы** физических объектов.

## Зачем это нужно?

- Тяжёлый якорь (увеличить массу)
- Лёгкие летающие объекты (уменьшить массу)
- Балансировка конструкций
- Копирование массы одного объекта на другой

## Как это работает внутри движка?

### Управление

| Кнопка | Действие |
|--------|----------|
| **ЛКМ** | Установить массу (из настройки `Value`) |
| **ПКМ** | Скопировать массу объекта в настройку `Value` |
| **R** | Сбросить массу к значению по умолчанию |

### Компонент PhysicalProperties

При установке массы создаётся / обновляется компонент `PhysicalProperties` (см. [12.04](12_04_MassOverride.md)):

```csharp
var mo = rb.GetOrAddComponent<PhysicalProperties>();
mo.Mass = mass;
mo.Apply();
```

`PhysicalProperties` — компонент из [фазы 12](12_04_MassOverride.md) (Components), который сохраняет переопределённую массу (вместе с `GravityScale` и `Health`) и применяет её к `Rigidbody.MassOverride`. Раньше для этого использовался отдельный компонент `MassOverride`; он был объединён с `PhysicalProperties`, чтобы все «надстроечные» физические параметры жили на одном компоненте и переживали дупликацию вместе.

### Сброс массы

```csharp
if ( mass <= 0f )
{
    rb.GetComponent<PhysicalProperties>()?.Destroy();
    rb.MassOverride = 0f;
}
```

При сбросе:
1. Удаляется компонент `PhysicalProperties` (с ним пропадают и сохранённые `GravityScale`/`Health`, если они были выставлены этим инструментом).
2. `MassOverride = 0` означает «использовать массу по умолчанию».

### Копирование массы

```csharp
[Rpc.Host]
private void CopyMass( Rigidbody rb )
{
    var mo = rb.GetComponent<PhysicalProperties>();
    Value = mo.IsValid() ? mo.Mass : rb.Mass;
}
```

Если у объекта есть `PhysicalProperties` — берём его. Иначе — фактическую массу из `Rigidbody`.

## Создай файл

📄 `Code/Weapons/ToolGun/Modes/Mass.cs`

```csharp
﻿﻿
[Icon( "🍔" )]
[ClassName( "mass" )]
[Group( "Tools" )]
public class Mass : ToolMode
{
	[Sync, Property, Title( "Mass (kg)" ), Range( 1, 250 ), Step( 0.5f )]
	public float Value { get; set; } = 100.0f;

	public override string Description => "#tool.hint.mass.description";
	public override string PrimaryAction => "#tool.hint.mass.set";
	public override string SecondaryAction => "#tool.hint.mass.copy";
	public override string ReloadAction => "#tool.hint.mass.reset";

	public override void OnControl()
	{
		var select = TraceSelect();
		if ( !select.IsValid() ) return;

		var rb = select.GameObject.GetComponent<Rigidbody>();
		if ( !rb.IsValid() ) return;

		if ( Input.Pressed( "attack1" ) ) SetMass( rb, Value );
		else if ( Input.Pressed( "attack2" ) ) CopyMass( rb );
		else if ( Input.Pressed( "reload" ) ) SetMass( rb, 0.0f );
		else return;

		ShootEffects( select );
	}

	[Rpc.Host]
	private void SetMass( Rigidbody rb, float mass )
	{
		if ( !rb.IsValid() || rb.IsProxy ) return;

		if ( mass <= 0f )
		{
			rb.GetComponent<PhysicalProperties>()?.Destroy();
			rb.MassOverride = 0f;
			return;
		}

		var mo = rb.GetOrAddComponent<PhysicalProperties>();
		mo.Mass = mass;
		mo.Apply();
	}

	[Rpc.Host]
	private void CopyMass( Rigidbody rb )
	{
		if ( !rb.IsValid() || rb.IsProxy ) return;

		var mo = rb.GetComponent<PhysicalProperties>();
		Value = mo.IsValid() ? mo.Mass : rb.Mass;
	}
}
```

## Что проверить?

1. Устанавливаем Value = 1 → ЛКМ на объект → он лёгкий (легко двигается)
2. Value = 250 → ЛКМ → тяжёлый
3. ПКМ на объект → Value обновляется
4. R → масса возвращается к стандартной
