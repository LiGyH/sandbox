# 00.06 — Настройки ввода (Input)

## Что мы делаем?

Настраиваем **клавиши управления** — какая кнопка за что отвечает.

## Как работает ввод в s&box?

Когда ты нажимаешь клавишу `W`, движок не знает что это значит «идти вперёд».  
Ему нужна **таблица привязок** (bindings), которая говорит: `W` → `"Forward"`.

Потом в коде мы будем писать: `if (Input.Down("Forward"))` — и это сработает  
когда игрок нажмёт `W` на клавиатуре (или стик на геймпаде).

## Разбор по группам

### Группа «Movement» (Движение)
| Действие | Клавиша | Что делает |
|----------|---------|------------|
| Forward | W | Идти вперёд |
| Backward | S | Идти назад |
| Left | A | Идти влево |
| Right | D | Идти вправо |
| Jump | Space | Прыжок |
| Run | Shift | Бег |
| Walk | Alt | Медленная ходьба |
| Duck | Ctrl | Присесть |

### Группа «Actions» (Действия)
| Действие | Клавиша | Что делает |
|----------|---------|------------|
| attack1 | ЛКМ | Основная атака / стрельба |
| attack2 | ПКМ | Дополнительная атака / прицел |
| Reload | R | Перезарядка |
| Use | E | Использовать (открыть дверь, подобрать) |

### Группа «Inventory» (Инвентарь)
| Действие | Клавиша | Что делает |
|----------|---------|------------|
| Slot1-Slot0 | 1-0 | Выбор слота оружия |
| SlotPrev | Mouse4 | Предыдущий слот |
| SlotNext | Mouse5 | Следующий слот |

### Группа «Other» (Прочее)
| Действие | Клавиша | Что делает |
|----------|---------|------------|
| View | X | Переключить вид камеры |
| Voice | V | Голосовой чат |
| Drop | G | Бросить оружие |
| Flashlight | F | Фонарик |
| Score | Tab | Таблица результатов |
| Chat | Enter | Открыть чат |
| Die | U | Самоубийство (респавн) |
| Spray | T | Спрей (граффити) |
| InspectMenu | C | Контекстное меню/инспектор |
| Undo | Z | Отмена последнего действия |
| SpawnMenu | Q | Меню спавна |

> **Как это работает внутри движка:**  
> Система ввода (`engine/Sandbox.Bind/`) регистрирует все действия из `Input.config`.  
> Каждый кадр она проверяет состояние клавиш через API операционной системы.  
> В коде ты обращаешься к ним через статический класс `Input`:
> ```csharp
> Input.Down("Forward")     // клавиша зажата
> Input.Pressed("Jump")     // клавиша только что нажата
> Input.Released("attack1") // клавиша только что отпущена
> ```
