# 00.04 — Настройки физики

## Что мы делаем?

Настраиваем **физический движок** — как быстро он считает столкновения и гравитацию.

## Что такое физика в игре?

Когда ты бросаешь предмет — он падает. Когда он ударяется об пол — он останавливается.  
За всё это отвечает **физический движок**. В s&box используется физика из Source 2.

## Шаг 1: Создай файл

Путь: `ProjectSettings/Physics.config`

Создай (или открой) этот файл и вставь:

```json
{
  "UseFixedUpdate": true,
  "SubSteps": 4,
  "FixedUpdateFrequency": 50,
  "MaxFixedUpdates": 2,
  "__guid": "327be197-6281-4b27-9b7f-5f406849093a",
  "__schema": "configdata",
  "__type": "PhysicsSettings",
  "__version": 1
}
```

## Что означает каждая строка?

| Поле | Значение | Объяснение |
|------|----------|------------|
| `UseFixedUpdate` | `true` | Использовать фиксированный шаг обновления (не зависит от FPS) |
| `SubSteps` | `4` | Физика считается 4 раза за один шаг (точнее, но медленнее) |
| `FixedUpdateFrequency` | `50` | 50 расчётов физики в секунду (совпадает с TickRate) |
| `MaxFixedUpdates` | `2` | Максимум 2 обновления физики за один кадр (защита от лагов) |

> **Как это работает внутри движка:**  
> Класс `PhysicsSettings` находится в ядре движка.  
> Каждый кадр игры, движок проверяет: «Прошло ли 1/50 секунды (20мс)?»  
> Если да — вызывается **FixedUpdate** для всех компонентов.
>
> `SubSteps: 4` означает, что каждый физический шаг делится на 4 мини-шага.  
> Это делает физику **точнее** — объекты не проходят сквозь друг друга.
>
> `MaxFixedUpdates: 2` — защита: если игра лагает и физика не успевает,  
> она не будет делать больше 2 шагов за кадр (иначе лаг только усилится).

## Почему 50 Hz?

- 50 обновлений в секунду — это **стандарт** для многих игр (Counter-Strike, Garry's Mod)
- Это баланс между **точностью** (чем больше — тем точнее) и **производительностью** (чем больше — тем больше нагрузка)
- Совпадает с `TickRate` в `.sbproj` — сервер и физика работают синхронно

## Результат

После этого этапа:
- ✅ Физика настроена на 50 Hz
- ✅ 4 подшага для точности
- ✅ Защита от лагов включена
